📝 Descripción del Proyecto

Este proyecto implementa un Sistema de Gestión de Metas Personales, desarrollado con ASP.NET Core MVC, Entity Framework Core y SQL Server.
El sistema permite al usuario organizar metas personales, crear tareas, asignar prioridades y controlar su progreso.


Examen II 2025

🎯 Objetivo del Sistema

Resolver el problema planteado en el examen:
El usuario tiene dificultades para priorizar tareas y visualizar avances debido al uso de notas dispersas y recordatorios básicos.
Este sistema centraliza metas y tareas, permitiendo una gestión clara y organizada.
